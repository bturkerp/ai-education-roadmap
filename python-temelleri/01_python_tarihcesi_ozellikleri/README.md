# Python Tarihçesi

## Python'un Doğuşu
Python, Guido van Rossum tarafından 1989 yılında Amsterdam'da geliştirilmeye başlandı. Guido, Hollanda doğumlu bir yazılım geliştiricidir.

## Kronolojik Gelişim

| Yıl | Olay | Önem |
|-----|------|------|
| 1989 | Geliştirme başladı | Noel tatilinde |
| 1991 | İlk sürüm (0.9.0) | alt.sources'ta yayınlandı |
| 1994 | Python 1.0 | İlk resmi sürüm |
| 2000 | Python 2.0 | List comprehensions eklendi |
| 2008 | Python 3.0 | Geriye dönük uyumsuz sürüm |
| 2020 | Python 2 desteği bitti | Artık sadece Python 3 |

## İlginç Bilgiler

### Python İsmi
Python adı, Guido'nun sevdiği "Monty Python's Flying Circus" adlı İngiliz komedi grubundan geliyor.

### Guido van Rossum
- **Doğum**: 31 Ocak 1956, Hollanda
- **Unvan**: Python'un Yaratıcısı
- **Kariyer**: Google (2005-2012), Dropbox (2013-2019)
- **Şu an**: Emekli

### Zen of Python
Python'un felsefesi Tim Peters tarafından yazılan 19 prensipten oluşur.

Önemli prensipler:
1. Okunabilirlik önemlidir
2. Basit olan karmaşık olandan iyidir
3. Açık seçik olan örtük olandan iyidir
