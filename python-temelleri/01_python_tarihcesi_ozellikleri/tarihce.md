# Python Tarihçesi

## Python'un Doğuşu
Python, Guido van Rossum tarafından 1989 yılında Amsterdam'da geliştirilmeye başlandı.

## Kronolojik Gelişim

### 1989 - Başlangıç
Guido van Rossum, Noel tatilinde yeni bir programlama dili üzerinde çalışmaya başladı.

### 1991 - İlk Sürüm
Python 0.9.0, Şubat 1991'de yayınlandı.

### 1994 - Python 1.0
İlk resmi sürüm piyasaya sürüldü.

### 2000 - Python 2.0
List comprehensions ve garbage collection eklendi.

### 2008 - Python 3.0
Geriye dönük uyumsuz sürüm yayınlandı.

### 2020 - Python 2 Desteği Sona Erdi
Python 2.x desteği resmen sona erdi.

## İlginç Bilgiler
